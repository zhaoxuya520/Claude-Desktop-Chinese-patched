关于本项目

本项目为 Claude Desktop for Windows 提供完整的中文界面汉化，覆盖 99% 以上的界面字符串。

采用双层汉化架构：

  • JSON 翻译文件覆盖 formatMessage 机制加载的字符串
  • JS 补丁 覆盖源码中硬编码的 title / label / placeholder 等

⚠️ 需要管理员权限 · 仅支持 Windows · Claude 更新后需重新安装

安装指南

1
下载汉化包 从 GitHub Releases 下载最新版 zip 并解压。
2
以管理员身份运行 右键 scripts/中文安装.bat → 以管理员身份运行。
3
重启 Claude Desktop 安装完成后关闭并重新打开 Claude Desktop，界面即显示为简体中文。

验证安装

运行以下命令检查汉化是否成功部署：

node scripts/_verify.js

全部 37 项检查通过（显示绿色 ✓）即为安装成功。

效果预览

安装后 Claude Desktop 界面将全面显示为简体中文：

  • 侧边栏 — 项目 / 会话 / 最近 / 已归档
  • 筛选器 — 状态 / 环境 / 分组/排序
  • 设置面板 — 全面中文化的 3P 配置
  • 搜索框 — 占位符提示中文
  • 弹窗提示 — 删除确认 / 导入 Operon 等

卸载

以管理员身份运行：

scripts/uninstall.ps1

或手动恢复备份文件（位于 %LOCALAPPDATA%\Claude-zh-CN-backup）。
